# Cloud-Native Chaos Engineering Platform for Kubernetes

A Chaos Engineering platform built using Kubernetes, FastAPI, Prometheus, Grafana and Minikube.

## Features

- Pod Kill Experiment
- CPU Stress Experiment
- Memory Stress Experiment
- Kubernetes API Integration
- Prometheus Monitoring
- Grafana Visualization

## Tech Stack

- Python
- FastAPI
- Kubernetes
- Docker
- Minikube
- Prometheus
- Grafana
